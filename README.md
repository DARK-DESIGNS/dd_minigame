Hello and welcome! This ressource provides you with an little minigame that is used in our own scripts and free for you to use!

Installation:
1.) put this line of code in your server.cfg: ensure dd_minigame
2.) Done

Support:
Yes! We Support all our scripts on our Discord(https://discord.gg/J5EEGnfhuE)

How to use:
You just need to put the Event-Trigger into your code and you will get and outcome as "true" or "false".
Here an example:

TriggerEvent("dd_minigame:start", function(outcome)
    if outcome == true then
        print("DONE")
    elseif outcome == false then
        print("NOT DONE")
    end
end)
